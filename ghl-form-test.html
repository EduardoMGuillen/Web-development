GHL Form Integration Test

Test Purpose

This page tests whether a GoHighLevel (GHL) form can be embedded and function properly outside of a GHL environment (e.g., on WordPress, static HTML, etc.).

What to Check:

  • Does the form load without errors?
  • Are all form fields visible and functional?
  • Can you submit the form successfully?
  • Do you receive confirmation after submission?
  • Are there any console errors in the browser's developer tools?
  • Does the form styling appear correctly?

Testing Instructions:

  1. Open this page in your browser
  2. Check if the form loads completely
  3. Try filling out and submitting the form
  4. Open browser developer tools (F12) and check the Console tab for any errors
  5. Test on different devices/browsers if possible
