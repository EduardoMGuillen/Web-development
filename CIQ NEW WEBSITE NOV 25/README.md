# CreativeIQ New Website - Figma Design Recreation

This folder contains the new CreativeIQ website built to match the Figma design while incorporating CreativeIQ's actual content and branding.

## File Structure

- **index.html** - Main website file with all sections

## Design Features

The website follows the Figma design specifications with:

### 1. Header/Navigation
- CreativeIQ logo with circular icon badge
- Navigation menu (Home, Services, About, Testimonials, Contact)
- Two call-to-action buttons (Get Quote, Start Project)

### 2. Hero Section
- "#1 Digital Marketing Agency" badge
- Headline: "Grow Your Business with Digital Excellence"
- Description text
- Two action buttons (Get Free Audit, Watch Demo)
- Hero image with data overlays (Monthly Growth, Conversion Rate)

### 3. Statistics Bar
- 500+ Successful Projects
- 98% Client Satisfaction
- 250% Average ROI

### 4. Services Section
- 6 service cards featuring CreativeIQ's actual services:
  - Search Engine Optimization
  - Pay-Per-Click Advertising
  - Social Media Marketing
  - Content Marketing
  - Analytics & Reporting
  - Email Marketing
- Custom solution call-to-action

### 5. About Section
- "Your Growth Partner in the Digital Age" headline
- Company description
- Three differentiators (Industry Recognition, Proven Results, Expert Team)
- Benefits list
- About image with overlay stats

### 6. Testimonials Section
- 6 testimonial cards with real client reviews
- Star ratings
- Client avatars and information
- Summary statistics (4.9/5 Rating, 500+ Clients, 98% Retention)

### 7. Contact Section
- Contact form with fields:
  - First Name, Last Name
  - Email, Phone
  - Company Name
  - Service Interested In (dropdown)
  - Monthly Budget Range (dropdown)
  - Message textarea
- Contact information sidebar:
  - Office location
  - Phone: (804) 651-2531

### 8. Footer
- Company information and description
- Contact details
- Social media icons
- Three columns of links (Services, Company, Resources)
- Footer bottom with copyright and legal links

## Color Scheme

Following the Figma design:
- **Primary Color**: Purple (#8B5CF6) - used for buttons, badges, accents
- **Text Dark**: #1F2937
- **Text Gray**: #6B7280
- **Background**: White (#FFFFFF) and Light Gray (#F9FAFB)
- **Footer**: Dark (#1F2937)

## Typography

- **Font Family**: Inter (Google Fonts)
- **Weights**: 300, 400, 500, 600, 700, 800

## Responsive Design

The website is fully responsive with breakpoints at:
- Desktop: 1280px max-width
- Tablet: 1024px
- Mobile: 768px and below

## Content Sources

- Branding and services from CreativeIQ's current website (creativeiq.marketing)
- Real testimonials from actual clients
- Contact information from CreativeIQ
- Service descriptions and pricing information from existing site
